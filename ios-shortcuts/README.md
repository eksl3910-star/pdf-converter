# 갤러리 전송 단축어

iPhone 홈 화면에서 실행 → 갤러리 선택 → 카카오톡 / iCloud / Google Drive·내 iPhone 저장

## 모바일 다운로드

1. 아래 링크에서 `gallery-transfer.shortcut` 다운로드
2. **파일** 앱 → 다운로드한 파일 탭 → **단축어 가져오기**
3. **설정 → 단축어 → 신뢰하지 않는 단축어 허용** (처음 한 번)
4. 단축어 **ⓘ → 홈 화면에 추가**

## 미리 만들 폴더 (파일 앱)

```
iCloud Drive/맛집 리스트/사진
iCloud Drive/맛집 리스트/동영상

내 iPhone/맛집 리스트/사진
내 iPhone/맛집 리스트/동영상
```

Google Drive는 저장할 때 폴더를 직접 선택합니다.

## 메뉴

| 선택 | 동작 |
|------|------|
| 카카오톡 나에게 보내기 | 공유 → 카카오톡 → 나에게 보내기 |
| iCloud Drive에 저장 | `맛집 리스트/사진` · `동영상` 자동 분류 |
| Google Drive / 내 iPhone | Google Drive 또는 내 iPhone 중 선택 |
