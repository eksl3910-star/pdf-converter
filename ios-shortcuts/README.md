# 갤러리 전송 단축어

iPhone 홈 화면에서 실행 → 갤러리 선택 → 카카오톡 / Google Drive / 내 iPhone 저장

## 모바일 다운로드

Mac에서 서명 후 설치 (iOS 15+):

1. `gallery-transfer.shortcut` 다운로드
2. Mac: `shortcuts sign -m anyone -i gallery-transfer.shortcut -o signed.shortcut`
3. iPhone 에어드롭 → **단축어 가져오기**
4. **ⓘ → 홈 화면에 추가**

## 미리 만들 폴더 (파일 앱)

```
내 iPhone/맛집 리스트/사진
내 iPhone/맛집 리스트/동영상
```

Google Drive는 저장할 때 폴더를 직접 선택합니다.

## 메뉴

| 선택 | 동작 |
|------|------|
| 카카오톡 나에게 보내기 | 공유 → 카카오톡 → 나에게 보내기 |
| Google Drive | `맛집 리스트/사진` · `동영상` 자동 분류 (설치 후 Drive 위치 1회 지정) |
| 내 iPhone | `맛집 리스트/사진` · `동영상` 자동 분류 |
